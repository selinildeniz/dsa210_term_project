# DSA210 Term Project – Selin Ildeniz (35847) ## Project Title What Makes
Customers Recommend a Clothing Product? ## Description This project analyzes
23,000 customer reviews from a women's fashion e-commerce platform to understand
what drives product recommendations. ## Dataset Women's E-Commerce Clothing
Reviews — Kaggle
https://www.kaggle.com/datasets/nicapotato/womens-ecommerce-clothing-reviews ###
How to reproduce 1. Download the dataset from the link above 2. Place the CSV
file inside a /data folder 3. pip install -r requirements.txt 4. Run notebooks in
/notebooks in order ## Timeline - March 17 – Repo created - March 31 – Proposal
submitted - April 14 – EDA and hypothesis testing - May 5 – ML methods - May 18 –
Final report
